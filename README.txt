GPU Sniper (Playwright-Version mit verbessertem Preisparser)

1. Voraussetzungen:
   - Node.js installiert: https://nodejs.org/
   - Playwright installieren:
     > npm install
     > npx playwright install

1.1. In Ordner Verzeichnis CMD öffnen und installieren -->
     > npm install playwright
     > npx playwright install

2. Starten:
   > node server.js

3. Browser öffnen:
   http://localhost:3000

4. Eine Query wie "RTX 3070" eingeben und auf "Angebote aktualisieren" klicken

Fix: Der Preisparser wurde verbessert und zeigt nun keine fehlerhaften Zahlen mehr an.
